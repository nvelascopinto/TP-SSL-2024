# TP4: Trabajo Práctico Integrador (TPI)
## Analizador léxico, sintáctico y semántico de C

### Ficha técnica
- **Estándar de C utilizado**: C89/C90
- **Compilador de C utilizado**: gcc
- **Versión mínima del compilador de C utilizada**: 11.4.0
- **Versión mínima de *flex* necesaria**: 2.6.4
- **Versión mínima de *bison* necesaria**: 3.8.2
- **Secuencia de comandos utilizados para compilar y ejecutar el programa**:
```bash
make all
./bin/tp4 'archivo.i'
```

```bash
make all
```
